# Privacy Policy – Ivy Shared Notes

**Last updated:** June 2026  
**Developer:** Dominik Mryka

---

## 1. Information We Collect

When you use Ivy Shared Notes, the following data is processed:

| Data | Purpose |
|---|---|
| Email address | Account registration and login |
| Display name | Shown within shared workspaces |
| Note content | Core functionality of the app |
| Device token (FCM) | Push notifications |
| Purchase information | Subscription and purchase management |
| Crash reports | Bug fixing and app stability |

---

## 2. How Your Data Is Stored

All data is processed and stored via **Google Firebase**:

- **Location:** Google Cloud – Europe (Frankfurt, Germany)
- **Services used:**
  - Firebase Authentication (login)
  - Cloud Firestore (notes and user profiles)
  - Firebase Cloud Messaging (push notifications)
  - Firebase Crashlytics (crash reporting)

Subscription and purchase management is handled by **RevenueCat** and **Google Play**.  
Their respective privacy policies apply: [RevenueCat Privacy Policy](https://www.revenuecat.com/privacy) · [Google Privacy Policy](https://policies.google.com/privacy)

---

## 3. Legal Basis

Data is processed based on your consent through use of the app (Art. 6(1)(a) GDPR) and for the performance of our service agreement (Art. 6(1)(b) GDPR).

---

## 4. Data Sharing

We do **not** sell your data. Data is only shared with the infrastructure providers listed above (Google, RevenueCat) solely for the purpose of operating the app.
